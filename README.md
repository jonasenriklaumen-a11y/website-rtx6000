# RTX PRO 6000 Blackwell — Showcase-Website

Eine professionelle One-Page-Website über die **NVIDIA RTX PRO 6000 Blackwell** Workstation-GPU — gebaut mit purem HTML, CSS und Vanilla JavaScript (keine Frameworks, keine Dependencies).

## ✨ Features

- **SVG Line Drawing on Scroll** — eine technische Zeichnung der GPU entsteht Linie für Linie beim Scrollen (`stroke-dasharray` / `stroke-dashoffset`)
- **Partikel-Netzwerk** im Hero (Canvas, an Schaltkreise angelehnt)
- **Scroll-Reveal-Animationen** via `IntersectionObserver`
- **Animierte Zähler** für die Spezifikationen
- **Hover-Effekte**: Spotlight, das dem Cursor auf Karten folgt, Glow-Buttons, animierte Nav-Unterstriche
- **Click-Effekte**: Material-Ripple auf Buttons/Karten plus grüne Klick-Funken auf der ganzen Seite
- **Scroll-Fortschrittsbalken** und animierte Benchmark-Balken
- Respektiert `prefers-reduced-motion`

## 🎨 Design

Dunkelgrün (`#0e3a20`), Schwarz (`#030805`), NVIDIA-Grün als Akzent (`#76b900`), weiße Texte. Schriften: Inter + JetBrains Mono.

## 🚀 Lokal starten

Einfach `index.html` im Browser öffnen — kein Build-Schritt nötig.
